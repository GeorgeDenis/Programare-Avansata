*Am creat 3 clase ControlPanel, ConfigPanel, DrawPanel
*In ControlPanel am creat 4 butoane Load, Save, Reset, Exit si i-am dat functionalitate butonului de Exit
*In ConfigPanel am creat 2 input-uri in care putem pune numarul de linii si probabilitatea ca o linie sa fie
intre 2 puncte
*In DrawPanel am creat o metoda paintComponent ce se foloseste de numarul de puncte si probabilitatea liniilor
pentru a face desenul
*In main mi-am declarat obiecte din cele 3 clase, le-am setat pozitia in pagina, si am dat functionalitate
butonului de Create new game folosind si un bloc try...catch.
*Am creat clasele Line, Dot ce modeleaza linii si puncte
*Am creat clasa Game ce este responsabila cu logica jocului si pentru a salva si a dat loat unui joc
*Clasa DrawPanel se foloseste de clasa Game, si este acum responsabila cu metodele ce coloreaza liniile,
reseteaza tabla recolorand toate liniile si pentru a exporta un PNG.
