*Am adaugat la modelul facut la Compulsory clasele Programmer, Designer si Network
*Programer si Designer mostenesc clasa Person. Acestea contin un Map cu relatiile lor cu celelalte persoane sau companii
*Clasa Network contine o lista cu noduri, de asemenea o metoda computeImportance ce afiseaza pe ecran importanta unui
nod concretizata in numarul de conexiuni cu celelalte obiecte, o metoda ce sorteaza nodurile in functie de importanta si
o metoda ce afiseaza pe ecran obiectele, importanta si relatiile lor.
